# BMI 計算機

祝大家維持運動與飲食好習慣，讓我們的 BMI 值都健康。

## 使用技術

- 配合 UI 設計師稿件
- 使用 JavaScript 判斷式實現數字計算與範圍。
- 結合 todolist 功能。
- 自行撰寫 RWD。
